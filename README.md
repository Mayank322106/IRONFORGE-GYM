# IRONFORGE-GYM
IRONFORGE GYM — Website A responsive single-page gym website built with pure HTML, CSS, and vanilla JavaScript. 
It features a dark, high-contrast design with a animated hero section, programs showcase, trainer profiles,
tiered membership pricing, a class schedule, testimonials carousel, and a contact form with validation.
Additional touches include a custom animated cursor, scroll-reveal animations, and live statistics counters. 
The entire project lives in a single HTML file, making it effortless to deploy on GitHub Pages, Netlify, or Vercel.
